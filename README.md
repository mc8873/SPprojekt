# Spletna stran Radovednež

Spletna stran za postavljanje in odgovarjanje vprašanj.
## Ideja
Uporabniki lahko dodajajo svoja vprašanja in odgovarjajo na vprašanja ki so jih postavili drugi uporabniki. Vsak uporabnik se lahko prijavi, kar mu omogoči ocenjevanje vprašanj/odgovorov, oznake odgovora kot dobrega in označevanje objav kot neprimerne. Vsako vprašanje/odgovor/komentar je mogoče oceniti z dobro ali slabo oceno (like/dislike). Stran omogoča filtriranje vprašanj in sicer lahko prikaže najnovejše objave najprej, ali pa najbolje ocenjene naprej. Filtriramo jih lahko tudi po datumu objave. Vprašanju lahko spraševalec doda oznake, ki predstavljajo teme v katere spada vprašanje.
## Komentarji
Največ časa sem porabil za ustrezno dodajanje komentarjev in odgovorov. Stran potrebuje še funkcionalnos ob pritisku "like/dislike" gumba.
